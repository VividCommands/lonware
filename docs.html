System Requirements

Minimum

  • Windows 10 (1903+) or Windows 11
  • Intel Core i5 / AMD Ryzen 5 or better
  • 8 GB RAM
  • Administrator account privileges
  • 200 MB free disk space
  • Active internet connection

Recommended

  • Windows 11 22H2 or later
  • Intel Core i7 / AMD Ryzen 7 or better
  • 16 GB RAM dual-channel
  • Dedicated GPU NVIDIA or AMD
  • SSD system drive
  • Wired internet connection
Note: PC products are Windows-only. Console Tweaks are platform-independent. All software requires an active internet connection for license verification on first launch.

Quick Start

01

Check Your Email

Download link and license key arrive at checkout email. Check spam folder if not received within 15 minutes.

02

Run as Administrator

Right-click and select Run as administrator. Required for PC optimization products to apply system-level changes.

03

Disable Antivirus Temporarily

Some antivirus programs flag optimization tools due to registry access. Temporarily disable during setup then re-enable.

04

Follow the Setup Guide

Every product ships with clear step-by-step documentation. Read fully before starting. No technical expertise required.

05

Restart Your PC

After applying optimizations, restart to ensure all changes take effect. Some settings only activate after a clean reboot.
